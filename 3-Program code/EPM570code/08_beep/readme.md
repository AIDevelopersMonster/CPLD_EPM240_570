# Проект: Генерация звукового сигнала с использованием ползунков ckey[1] ~ ckey[4]

## Описание

Этот проект реализует модуль **beep**, который генерирует звуковой сигнал на основе тактового импульса (clk) с возможностью включения/выключения с помощью ползунков `ckey[1]`, `ckey[2]`, `ckey[3]`, и `ckey[4]`.

- **Ползунок `ckey[4]`** управляет включением/выключением зуммера.
- **Ползунок `ckey[1]`** включает эффект звука с частотой 1 кГц.
- **Ползунок `ckey[2]`** включает эффект звука с частотой 763 Гц (Меандр).
- **Ползунок `ckey[3]`** включает эффект звука с частотой 191 Гц (Меандр).
- Управление **PNP транзистором**, база которого подтянута к питанию.

## Особенности

- Использование четырёх ползунков для управления звуковыми эффектами.
- Логика генерации звука с использованием тактового сигнала (например, 50 МГц).
- Управление зуммером через **PNP транзистор**.
- Возможность изменения частот для каждого ползунка для создания различных звуковых эффектов.

## Описание модуля

1. **Входной сигнал**:
   - `clk`: Тактовый сигнал, используемый для синхронизации работы модуля.
   - `ckey[1]`, `ckey[2]`, `ckey[3]`, `ckey[4]`: Входной сигнал для управления состоянием зуммера и звуковыми эффектами.

2. **Выходной сигнал**:
   - `beep`: Выходной сигнал для управления PNP транзистором и зуммером.

3. **Регистры**:
   - `beep_r`: Регистр, который хранит состояние выходного сигнала.
   - `count`: Счётчик, который делит частоту входного тактового сигнала для формирования нужной частоты выходного сигнала.

## Принцип работы

1. Если `ckey[4] == 1`, зуммер выключен независимо от других ползунков.
2. Если `ckey[1] == 1`, создается звук с частотой 1 кГц.
3. Если `ckey[2] == 1`, создается звук с частотой 763 Гц (Меандр).
4. Если `ckey[3] == 1`, создается звук с частотой 191 Гц (Меандр).
5. Если все ползунки выключены, зуммер не генерирует звук.

## Установка

1. Скачайте или клонируйте репозиторий:
    ```bash
    git clone https://github.com/AIDevelopersMonster/CPLD_EPM240_570.git
    ```

2. Добавьте файл `beep_with_ckey_pins_pnp.v` в свой проект Verilog.

3. Включите модуль в топовый уровень проекта:
    ```verilog
    module top(
        input clk,      // Входной тактовый сигнал
        input [4:1] ckey,   // Входной сигнал ckey[1] ~ ckey[4]
        output beep     // Выходной сигнал для звука
    );

    // Подключение модуля beep
    beep beep_inst (
        .clk(clk),      // Тактовый сигнал
        .ckey(ckey),    // Сигнал ползунков ckey[1] ~ ckey[4]
        .beep(beep)     // Выходной сигнал
    );

    endmodule
    ```

4. Скомпилируйте и загрузите проект на ваше устройство (FPGA или CPLD).

## Примечания

- Этот проект использует **синхронизацию по положительному фронту тактового сигнала** для работы счётчика.
- Частота звукового сигнала зависит от настроек счётчика и может быть легко изменена путём модификации числа разрядов счётчика или логики генерации.
- Подключите выходной сигнал `beep` к базе PNP транзистора для управления зуммером.
