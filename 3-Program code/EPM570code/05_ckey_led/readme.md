
# ckey_led.v - Модуль управления светодиодами с помощью ползунковых переключателей

## Описание модуля

Модуль `ckey_led.v` предназначен для управления 4 светодиодами с помощью 4 ползунковых переключателей. Каждое положение переключателя будет управлять состоянием соответствующего светодиода. Когда переключатель в положении "включен" (значение 1), соответствующий светодиод будет включен. Если переключатель в положении "выключен" (значение 0), то светодиод будет выключен.

## Входы и выходы

- **Вход `ckey[4:1]`**: 4-битный вектор, представляющий состояние 4 ползунковых переключателей.
  - Каждый бит соответствует одному ползунковому переключателю.
  - Если бит = 1, то соответствующий переключатель включен (и светодиод загорается).
  - Если бит = 0, то соответствующий переключатель выключен (и светодиод выключается).

- **Выход `led[1:4]`**: 4-битный вектор, представляющий состояние 4 светодиодов.
  - Каждый бит управляет одним из светодиодов. Значение 1 — светодиод включен, значение 0 — светодиод выключен.

## Принцип работы

Когда пользователи изменяют положение ползунков на плате, значения вектора `ckey` изменяются, и это напрямую влияет на состояние светодиодов. В случае, если ползунковый переключатель на позиции включен, соответствующий светодиод включается. Если переключатель на позиции выключен, светодиод выключается.

### Пример:

- Если `ckey = 4'b0101`, то светодиоды `led[1]` и `led[3]` будут включены, а остальные выключены.
- Если `ckey = 4'b1111`, то все светодиоды будут включены.

## Структура модуля

```verilog
module ckey_led(
  input  [MSB_reg:1] ckey,  // Входной сигнал ckey, состоящий из 4 бит (ckey[1] ~ ckey[4])
  output [1:4] led         // Выходной сигнал led, состоящий из 4 бит (led[1] ~ led[4])
);
parameter MSB_reg = 4;

assign led = ckey; // Присваивание значений из входа ckey на выход led

endmodule
```

## Описание синтаксиса:

1. **Входные и выходные порты**: 
   - Входной сигнал `ckey` представляет собой 4 бита, каждый из которых контролирует положение одного ползункового переключателя.
   - Выходной сигнал `led` также представляет собой 4 бита, каждый из которых управляет состоянием одного светодиода.

2. **Присваивание значений**:
   - Оператор `assign` используется для передачи значения с входа `ckey` на выход `led`. Таким образом, состояние каждого переключателя напрямую влияет на состояние соответствующего светодиода.

## Использование

1. Подключите 4 ползунковых переключателя к входам `ckey[1]` - `ckey[4]` на вашей плате.
2. Подключите 4 светодиода к выходам `led[1]` - `led[4]`.
3. Модуль будет автоматически управлять светодиодами в зависимости от положения переключателей.
